Node Crawler
----------------------

1.  Open terminal and run
```shell script
#installation   
yarn install

#run
yarn start -c <crawler>

#note
Usage: -c <crawler>

Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -c, --crawler  Crawler name, eg: shopify-theme             [string] [required]

```
